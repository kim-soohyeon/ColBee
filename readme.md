# ColBee (콜비): 혼자 사는 사람을 위한 냉장고 관리 서비스

**ColBee**는 "버리지 않는 냉장고"라는 핵심 메시지를 바탕으로, 혼자 사는 사람들이 식재료를 부지런히 관리하고 음식물 쓰레기를 최소화할 수 있도록 돕는 스마트 냉장고 관리 서비스입니다.

## 🚀 주요 구현 기능

구현 계획서 및 `readme.md` 명세를 기반으로 다음과 같은 핵심 기능들이 구현되었습니다.

### 1. 프론트엔드 (React + Vite)
- **메인 대시보드 (Home)**: 냉장고의 전반적인 상태(안전, 주의, 긴급, 만료)를 시각적으로 요약.
- **콜비 마스코트 시스템**: 유통기한 임박도에 따라 콜비의 표정(😊, 😐, 😰, 😢)과 애니메이션이 실시간으로 변화.
- **식재료 관리 (List)**: 등록된 식재료의 목록과 상태를 확인.
- **식재료 등록 (Register)**: 제품명과 유통기한을 입력하여 새로운 식재료를 냉장고에 등록.
- **장보기 리스트 (Shopping)**: 소비 패턴과 유통기한 정보를 바탕으로 자동 생성된 추천 품목 확인 및 관리.
- **월간 리포트 (Analytics)**: 소비 통계, 쓰레기 배출률, 절약액 및 콜비의 맞춤형 조언 제공.

### 2. 백엔드 핵심 로직 (Node.js/JS)
- **Mascot Service**: 식재료 데이터 기반 감정 상태 및 애니메이션 결정 엔진.
- **Notification Service**: 카테고리별(유제품, 채소, 육류 등) 맞춤형 알림 메시지 생성.
- **Analytics Service**: 월간 데이터 기반의 지능형 통계 및 개선 조언 생성.
- **Ingredient/Shopping Service**: 식재료 등록 및 장보기 리스트 자동 생성 로직.

### 3. 데이터베이스 (PostgreSQL)
- `database/schema.sql`: 사용자, 냉장고, 식재료, 장보기 리스트, 통계, 알림 테이블을 포함한 통합 스키마 설계 완료.

## 🛠 실행 방법

### 프론트엔드 실행
1. `frontend` 디렉토리로 이동합니다.
   ```bash
   cd frontend
   ```
2. 필요한 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```
4. 브라우저에서 `http://localhost:5173` 접속 후 콜비 서비스를 확인합니다.

### 백엔드 로직 확인
백엔드 로직은 현재 `backend/services/` 내부의 각 서비스 디렉토리에 핵심 알고리즘(`.js`)이 구현되어 있습니다. 실제 API 서버를 구동하기 위해서는 각 서비스별로 엔트리 포인트 설정이 필요합니다.

## 📁 프로젝트 구조
- `frontend/`: React 웹 애플리케이션
- `backend/services/`: 핵심 마이크로서비스 로직
- `database/`: SQL 스키마 명세
- `docs/`: 프로젝트 관련 문서
- `RUN_GUIDE.md`: 상세 실행 안내
